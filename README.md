## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/portfolio.git
```

2. Navigate to the project 
```sh
cd portfolio
```
3. Open index.html in your browser


## How to use

1. Open the website in your browser
2. Navigate through the sections:
    - Home: view information about me.
    - Skills: explore the technologies and tools I use.
    - Experience: see my work experience and professional journey
    - Projects: explore the projects and interact with them.
    - Contact: get in touch for collaborations, questions, or opportunities. 
3. If needed, switch between dark and light themes
